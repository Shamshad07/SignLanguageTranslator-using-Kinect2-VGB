Constructor for computing DTW matrix

Parameters
dimThe dimension of the array. It must be 12(X and Y cordinates) to compute six hand joints
thresholdThe threshold parameter for sequence matching in DTW. The less its value, the more similar the sequence.
firstThresholdIt is boundary condition for the maximum distance between sequences. For more accuracy, less value is required
maxSlopeIt is used to avoid mapping very steep slopes with sequences. For more accuracy, less value is required
minimumLengthMinimum length is constraint in sequence before it is matched.
